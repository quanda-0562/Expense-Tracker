## ⚠️ Important Notes

### Before You Start
- [ ] Have Node.js 18+ installed
- [ ] Have a Supabase account (free tier available)
- [ ] Have Git installed
- [ ] Read [QUICKSTART.md](./QUICKSTART.md) first

### Before Deploying
- [ ] Set up Supabase database (use [SUPABASE_SETUP.md](./SUPABASE_SETUP.md))
- [ ] Test signup/login locally
- [ ] Test transaction creation
- [ ] Verify all environment variables are configured
- [ ] Follow [DEPLOYMENT.md](./DEPLOYMENT.md) exactly

### Security
- [ ] Never commit `.env.local` to Git
- [ ] Keep Supabase credentials secret
- [ ] Enable Row Level Security (done in setup)
- [ ] Use strong database passwords
